# 🌐 3D Developer Portfolio

A **modern and immersive 3D developer portfolio** built with **React**, **Three.js**, **Tailwind CSS**, and **Framer Motion** – designed to showcase your skills, experience, and projects in a visually engaging way.

---

## 📸 Live Demo  
🔗 [View Portfolio](https://portfolio-xi-virid-26xa0s31cf.vercel.app/)

---

## ✨ Features

- 🎯 Animated 3D models using `@react-three/fiber` & `@react-three/drei`
- 🧑‍💼 Sections for About, Skills, Work Experience, Projects, and Contact
- 💼 Professional timeline showcasing internships & projects
- 🌗 Dark mode and responsive design
- 🎮 Interactive skill stack with 3D & static icons
- ⚡ Smooth scroll and hover animations via Framer Motion
